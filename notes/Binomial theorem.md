## Middle term

```
(x + y)^n
```

No of terms = n + 1

- 1 Middle term
n == even {
  ((n + 1) + 1) / 2
}

- 2 Middle term
n == odd { 
  (n + 1) / 2
  (n + 3) / 2 
}

Middle term contains highest binomial coefficient.
